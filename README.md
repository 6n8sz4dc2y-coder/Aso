# RRG Group Dashboard

Static GitHub Pages dashboard for weekly RRG performance reporting.

## Deploy

Upload these files to the root of the GitHub repository and enable GitHub Pages from `main` / root:

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `CHANGELOG.md`
- `LICENSE`
