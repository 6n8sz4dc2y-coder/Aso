# Dealer Registration Dashboard

First prototype dashboard using Q2 Toyota registration data.

## What it does
- Shows group registration performance
- Shows centre and CDA performance
- Ranks centres by % to target
- Highlights ahead / on track / behind

## Files
- `index.html` - the dashboard
- `q2_registration_dashboard_data.csv` - source data used for the prototype

## Run locally
Open `index.html` in your browser.

## Publish with GitHub Pages
1. Create a new GitHub repository.
2. Upload these files.
3. Go to Settings > Pages.
4. Under Build and deployment, choose `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save. GitHub will give you a live website link.
